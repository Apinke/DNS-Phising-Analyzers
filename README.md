## About

A sophisticated DNS phishing domain analyser that combines **Traditional DNS Heuristics** with **Gemma 3 Pro's** contextual reasoning to detect phishing domains before campaigns go live.

## Features

- 🔍 **Lexical Analysis** — Detects Punycode, phishing keywords, and excessive hyphens
- 📊 **Threat Probability Score** — Visual risk meter with severity classification
- 🧠 **Gemma 3 Pro Verdict** — AI-powered strategic reasoning and recommendations
- 📐 **Brand Similarity Detection** — Levenshtein distance analysis against monitored brands
- 🛡️ **SPF/DMARC Simulation** — Email authentication posture assessment
- ⚡ **Dark Staging Detection** — Identifies pre-launch phishing infrastructure

## How to Use

1. Enter a **suspicious domain** (e.g. `paypa1-secure-auth.com`)
2. Enter the **brand domain** to compare against (e.g. `paypal.com`)
3. Click **Initiate Global DNS Audit**
4. Review the Threat Score, Structural Indicators, and Gemma 3 Pro Verdict

## Threat Scoring

| Score | Severity | Action |
|---|---|---|
| 0–30 | Nominal | Monitor for changes |
| 31–70 | Suspicious | Investigate further |
| 71–100 | Critical | Immediate blacklisting |
