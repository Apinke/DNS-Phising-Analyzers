 The standalone DNS Phishing Analyzer (built for a backend deployment like Google Cloud Run) uses an Intrinsic Analysis Architecture. Unlike comparative models that require a known "base brand" (like paypal.com) to measure against, this system evaluates a domain purely in a vacuum by looking for inherent structural and infrastructural anomalies.
The architecture is divided into three primary layers: the API Gateway, the Tri-Modular Analysis Engine, and the Threat Synthesis Engine.
